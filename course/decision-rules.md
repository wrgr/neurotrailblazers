# Editorial Decision Rules

## Source precedence (default)
1. `frompat` (most mature; preserve key info and examples)
2. `neuroAI` (stable but narrower; preserve technical claims and citations)
3. `assets_outreach` (historical/contextual; use selectively and modernize wording)

## Conflict resolution
- If `frompat` conflicts with other sources on morphology/proofreading interpretation,
  retain `frompat` as primary and annotate alternatives in notes.
- If `neuroAI` conflicts with older outreach framing, keep `neuroAI` for methods/tooling
  and move outreach material to historical context callouts.
- Mark uncertain statements as `VERIFY` and queue for source check.

## Rewrite constraints
- Keep facts and core teaching intent; rewrite tone/structure for learner clarity.
- Separate historical claims (e.g., dated 2021 context) from current framing.
- Preserve attribution/citation lineage for every reused figure.

## Output constraints
- Every unit doc must include:
  - website-ready summary blocks
  - slide-ready sequence
  - selected figure candidates with attribution notes
  - open questions / verification items
