# Deck Spec: 05 Neuronal Ultrastructure

## Session
- Duration: 75 + 20 min
- Outcome: cue-based annotation sheet with confidence labels
- Primary source: `course/source-ingest/microns-proofreading-neuronal-ultrastructure-111821.pptx.md`
- Figure source: `course/units/figures/05-neuronal-ultrastructure-selected-v1.md`

## Slide production blocks
1. Title and task framing
- Key message: ultrastructure interpretation is a reproducible decision process, not visual intuition.
- Figure: `FIG-RIV-ULTRA-S04-01`
- Instructor note: define evidence tiers up front.

2. Compartment map refresher
- Key message: process identity begins with compartment-level anatomy.
- Figure: `FIG-RIV-ULTRA-S08-01`
- Instructor note: ask learners to label soma/shaft/spine candidates live.

3. Synapse-identification cues
- Key message: vesicle clusters + active zone + PSD context are evaluated together.
- Figure: `FIG-RIV-ULTRA-S09-01`
- Instructor note: require at least two converging cues before a hard call.

4. Organelle context for annotation
- Key message: mitochondria, ER, and microtubules reduce ambiguity.
- Figure: `FIG-RIV-ULTRA-S10-01`
- Instructor note: show one false-positive cue and how context corrects it.

5. Comparative ultrastructure
- Key message: similar-looking profiles diverge when local neighborhood is included.
- Figure: `FIG-RIV-ULTRA-S14-01`
- Instructor note: emphasize multi-slice review requirement.

6. Ambiguity handling protocol
- Key message: uncertainty labels preserve scientific validity.
- Figure: `FIG-RIV-ULTRA-S20-01`
- Instructor note: model a provisional label workflow.

7. Advanced case review
- Key message: difficult regions need explicit decision logs.
- Figure: `FIG-RIV-ULTRA-S24-01`
- Instructor note: compare two annotator interpretations and adjudicate.

8. Synthesis panel
- Key message: cue triangulation scales to complex scenes when procedure is consistent.
- Figure: `FIG-RIV-ULTRA-S30-01`
- Instructor note: close with decision checklist learners can reuse.

9. Failure modes and anti-patterns
- Key message: single-slice calls, single-cue dependence, and forced certainty drive major errors.
- Figure: reuse `FIG-RIV-ULTRA-S20-01`

10. QC metrics and auditability
- Key message: inter-rater agreement and unresolved-case rates are operational quality indicators.
- Figure: no new figure required (table slide).

11. Activity instructions
- Key message: annotate one panel with confidence + alternate hypothesis.
- Figure: `FIG-RIV-ULTRA-S24-01`

12. Debrief and bridge to Unit 06
- Key message: process-type classification extends this cue logic.

## Assessment
- Prompt: annotate one panel and justify decisions with three independent cues.
- Rubric: cue quality, coherence, uncertainty handling.
