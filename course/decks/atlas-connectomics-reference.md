# Deck Spec: Atlas Connectomics Reference

## Session
- Duration: 55 + 20 min
- Outcome: three curated atlas entries with full metadata
- Figure source: `course/units/figures/atlas-connectomics-reference-selected-v1.md`

## Slide plan
1. Title and atlas role
   - Content: operational reference, not passive bibliography.
2. Metadata schema
   - Content: citation, modality, scale, access, maturity, limits.
3. Workflow-stage indexing
   - Content: acquisition, reconstruction, proofreading, analysis.
4. Evidence grading
   - Content: concept, validated prototype, production-validated.
5. Reproducibility criteria
   - Content: code/data/protocol availability.
6. Link hygiene policy
   - Content: stale-link checks and replacement policy.
7. Historical resource handling
   - Content: retain context, flag as historical.
8. Worked curation example
   - Content: paper + dataset + tool mapping.
9. Governance model
   - Content: roles, review cadence, ownership.
10. Failure modes
    - Content: duplicate entries, incomplete metadata, stage imbalance.
11. Curation activity
    - Content: author one entry with limitations statement.
12. Wrap and maintenance cycle
    - Content: update protocol and accountability.

## Assessment
- Prompt: produce one complete atlas entry mapped to a track unit.
- Rubric: completeness, relevance, and limitation clarity.
