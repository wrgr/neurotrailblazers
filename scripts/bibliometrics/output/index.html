Data Sources
Primary: OpenAlex API (academic paper metadata)
Coverage: 1977–2026
Scope: EM connectomics + related fields
Deduplication: Preprints merged with published versions
Metrics Explained
In-degree: How many papers cite this (foundational signal)
Out-degree: How many this paper cites (synthesis signal)
PageRank: Network importance via link structure
K-core: Local network density
Composite Score: Weighted combination of above
Paper Roles
Landmark: High in-degree, few references (C. elegans, Drosophila)
Foundational: Frequently cited, seminal work
Synthesis: High out-degree, integrate field knowledge
Methods: Reference 50+ papers, enable others
Applied: Use methods on biology
Key Insights
• Methods papers (high out-degree) critical to field
• Citation networks more robust than concept-based clustering
• 571 preprints remain as legitimate research
• 7,503 final papers after deduplication
• Field evolved: Theory (1977–2000) → Methods (2000–2015) → Connectomes (2020+)
